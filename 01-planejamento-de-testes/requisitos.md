# Requisitos – Automation Practice

## 1. Cadastro
- Campo e-mail deve ser válido
- Campos obrigatórios devem estar preenchidos
- Senha deve ter no mínimo 6 caracteres
- Email não pode estar duplicado

## 2. Login
- Email e senha obrigatórios
- Email deve ser válido
- Autenticação só ocorre com credenciais válidas

## 3. Busca
- Deve permitir pesquisa por nome
- Deve retornar produtos relacionados
- Não deve quebrar com inputs inválidos

## 4. Produto
- Tela deve exibir imagem, preço, descrição
- Deve permitir adicionar ao carrinho

## 5. Carrinho
- Deve permitir adicionar/remover produtos
- Deve atualizar subtotal e total
- Deve validar quantidade

## 6. Checkout
- Requer login
- Deve seguir etapas: carrinho → endereço → pagamento → confirmação

## 7. Contato
- Deve enviar mensagem com nome, email e assunto
