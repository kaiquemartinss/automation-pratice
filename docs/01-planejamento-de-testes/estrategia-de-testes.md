# Estratégia de Testes

## 1. Abordagem Geral
Os testes serão executados manualmente, seguindo cenários funcionais e exploratórios, cobrindo validações críticas primeiro.

---

## 2. Tipos de Teste
- **Funcionais**
- **UI/UX**
- **Exploratórios**
- **Regressão**

---

## 3. Prioridades
| Prioridade | Critério |
|-----------|----------|
| Alta | Impacta compra, cadastro, login |
| Média | Funcionalidades secundárias |
| Baixa | Textos, responsividade, estética |

---

## 4. Ambiente de Testes
**URL:** https://www.automationpratice.com.br/  
**Navegador principal:** Google Chrome

---

## 5. Execução
Os testes seguirão:
1. Análise exploratória inicial  
2. Execução dos casos de teste  
3. Abertura de bugs  
4. Reexecução para regressão
